# Pre-Wedding Guys Trip - City Picker

Interactive city comparison tool for planning a guys trip before a Dolomites wedding (Aug 25-31, 2026).

## Cities

- **Barcelona** - Beach + nightlife + culture ($1,250-1,800/person)
- **Madrid** - Best value, 6am nightlife, tapas ($1,100-1,600/person)
- **Dubrovnik** - Walled city, Adriatic, Montenegro day trip ($1,300-1,900/person)
- **Monaco / Nice** - Casino, yachts, beach clubs ($1,300-1,800/person)
- **Switzerland** - Paragliding, Alps, adrenaline ($1,500-2,200/person)

Each option includes: RT flight from SFO, 2 nights hotel (3.5-4 star), transport to Venice, 1 night Venice, and bus to Dolomites.

## Deploy

Static HTML. Push to GitHub, connect to Vercel, done.

```bash
git init
git add .
git commit -m "city picker"
git remote add origin git@github.com:YOUR_USERNAME/pre-wedding-picker.git
git push -u origin main
```

Then import in Vercel dashboard or:

```bash
npx vercel
```

## Stack

Zero dependencies. Vanilla HTML/CSS/JS. One file.
